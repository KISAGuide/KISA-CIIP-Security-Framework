# CIIP Security Framework

CIIP Security Framework는 **2026년 한국인터넷진흥원(KISA) 「주요정보통신기반시설 기술적 취약점 분석·평가방법 상세가이드」를 준거로 설계된** 기술적 보안 점검 및 자동화 지원 프레임워크입니다.

---

## 주요 기능

- KISA 기술적 취약점 분석·평가 가이드에 기반한 점검 기준 제공
- 반복 가능하고 재현성 있는 보안 점검(Assessment) 체계 수립
- 취약점 점검 결과에 대한 증적(Evidence) 및 판단 근거 정형화
- 내부 보안 점검, 외부 감사, 관리·감독기관 제출 자료 대응 지원
- 수작업 중심 점검의 자동화 및 효율성 향상

---

## Reference & Compliance

본 프로젝트는 다음 가이드를 **참고 기준(reference)** 으로 활용합니다.

- 한국인터넷진흥원(KISA)
  - 「2026 주요정보통신기반시설 기술적 취약점 분석·평가방법 상세가이드」

본 프레임워크는 상기 가이드의 구조, 점검 영역, 보안 통제 관점을 참고하여 설계되었으며, 실제 점검 항목의 적용, 평가 기준, 취약/양호 판단은 **각 기관의 정책, 운영 환경, 위험 수용 수준에 따라 조정되어야 합니다.**

---

## 점검 대상

CIIP Security Framework는 다음 기술적 보안 점검 영역을 지원합니다.

- Unix / Linux 서버
- Windows 서버
- 웹 서비스 및 웹 애플리케이션
- 네트워크 장비 및 보안 장비
- DBMS
- 가상화 환경
- 클라우드 환경
  - 계정 관리
  - 권한 관리
  - 가상 리소스 관리
  - 운영 관리

※ 점검 항목은 KISA 가이드 개정 및 IT 환경 변화에 따라 지속적으로 업데이트됩니다.

---

## 주요 고객

- 주요정보통신기반시설 관리기관
- 공공기관 및 공기업 보안 담당자
- 기업 내부 보안 점검 및 감사 대응 조직
- 취약점 분석·평가 수행 인력
- KISA 가이드 기반 자동화 도구 또는 보안 프레임워크를 구축하려는 개발자

본 프로젝트는 **보안 점검 및 내부 평가 지원을 위한 도구**이며, 실제 침해사고 발생 여부, 법적 책임, 행정적 제재 여부에 대한 **공식 판단 또는 규제 기관의 결정을 대체하지 않습니다.**

본 프로젝트를 사용함으로써 발생하는 모든 책임은 사용자에게 있으며,  운영 환경에 적용하기 전 충분한 검토와 검증을 권장합니다.
