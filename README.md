# CIIP Security Framework

CIIP Security Framework는 **2026년 한국인터넷진흥원(KISA) 「주요정보통신기반시설 기술적 취약점 분석·평가방법 상세가이드」를 준거로 설계된** 기술적 보안 점검 및 자동화 지원 프레임워크입니다.

---

## 주요 기능

### 1. 기술 취약점 자동 진단
- 계정 및 권한 관리
- 서비스 및 데몬 설정
- 패치 및 업데이트 상태
- 네트워크 접근 통제
- 로그 및 감사 설정
- 보안 위협 탐지 항목

### 2. 항목별 모듈화 구조
- 진단 항목 단위로 독립적인 스크립트 모듈 구성
- 항목 추가/삭제/교체 가능
- 장기 유지보수 및 기준 변경 대응 용이

### 3. XML 형식 진단 리포트 출력
- 항목별 판정 결과(PASS / FAIL / NA / ERROR)
- 위험 등급(상 / 중 / 하)
- 증빙(evidence) 및 판정 근거(reason) 포함
- 외부 시스템 연계 및 후처리 가능

### 4. 항목별 조치 가이드 문서 제공
- 진단 항목과 **1:1 매핑**
- 단순 권고가 아닌 **실행 가능한 수준**의 가이드
- 실제 명령어, 설정값, 경로, 확인 방법 포함
- Word(.docx) 형식으로 제공

---

## 조치 가이드 문서 제공

각 조치 가이드는 아래 항목을 필수로 포함합니다.

- 항목 ID (KISA 기준)
- 항목명 및 위험도
- 취약 판정 기준
- 양호 판정 기준
- 조치 방법 (OS/플랫폼/버전별)
- 조치 후 확인 방법 (명령어 및 기대 결과)
- 주의사항 (서비스 영향, 예외 처리)
- 참고 기준 (KISA 가이드 페이지, 법령, CVE 등)



## Reference & Compliance

본 프로젝트는 한국인터넷진흥원(KISA)에서 발표한 "2026 주요정보통신기반시설 기술적 취약점 분석·평가방법 상세가이드" 를 준수합니다.

## 점검 대상

CIIP Security Framework는 다음 기술적 보안 점검 영역을 지원합니다.

- Unix / Linux 서버
- Windows 서버
- 웹 서비스 및 웹 애플리케이션
- 네트워크 장비 및 보안 장비
- DBMS
- 가상화 환경
- 클라우드 환경
  - 계정 관리
  - 권한 관리
  - 가상 리소스 관리
  - 운영 관리


## 대상 플랫폼

### DBMS

### WEB / WAS
- WebtoB
- JBoss
